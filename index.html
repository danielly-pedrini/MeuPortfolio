Projetos

Aqui estão alguns projetos que desenvolvi até agora:

  • Tela de Login com HTML e CSS - Criação de uma página de login simples utilizando HTML e CSS.
  • Alterações nas Cores de Fundo com HTML, CSS e JavaScript - Projeto onde o usuário pode alterar as cores de fundo de uma página usando JavaScript.
  • To-Do List - Desenvolvido com HTML, CSS e JavaScript. Criando uma aplicação web interativa para gerenciamento de tarefas. A aplicação permite adicionar, marcar como concluídas e remover tarefas, com armazenamento local.
  • Projeto FP Selection - Criação de um site interativo sobre carros, incluindo animações feitas com CSS e JavaScript.
  • Calculadora Básica - Uma calculadora web interativa construída com HTML, CSS e JavaScript puro, oferecendo operações matemáticas básicas e funcionalidades adicionais.
  • Carousel de Sabores de Refrigerante - Um projeto interativo que apresenta diferentes sabores de refrigerante, destacando cada variante através de uma interface dinâmica.
  • Previsão do Tempo - Um aplicativo simples que exibe a previsão do tempo de qualquer cidade em tempo real. Ele utiliza a API OpenWeather para buscar informações como temperatura, descrição do clima e umidade.
  • Relogi Digital - Este projeto apresenta um relógio digital interativo, desenvolvido utilizando HTML, CSS e JavaScript. Ele exibe a hora atual de maneira dinâmica e atualizada em tempo real. O design do relógio é simples e moderno, com uma interface que facilita a leitura das horas, minutos e segundos.
  • Galeria de Imagens - Este projeto é uma galeria de imagens responsiva que apresenta várias fotos organizadas em um layout elegante. A galeria permite que o usuário visualize as imagens de forma interativa.
